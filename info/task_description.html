Enunciado

Pedro, um editor de texto, precisa implementar uma funcionalidade de "desfazer" e "refazer" em seu software. Ele quer usar uma pilha para armazenar as ações realizadas e poder desfazê-las ou refazê-las. Crie uma função que simule as ações de desfazer e refazer em um editor de texto. A função deve permitir adicionar ações, desfazer a última ação e refazer a ação desfeita.

Entrada: Uma lista de comandos, onde cada comando é uma string: "fazer ", "desfazer", "refazer".

Saída: A lista de ações após a execução de todos os comandos.
